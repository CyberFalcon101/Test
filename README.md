# TryCrackMe Level 1 Write-Up
Welcome! This is an official write-up for the CTF
[TryCrackMe Level 1](https://tryhackme.com/room/trycrackmelevel1), made by the room's creator 
[CyberFalcon101](https://tryhackme.com/p/CyberFalcon101). 
Note that although the steps to find the answers are given, the actual solutions are not provided.
## Intro
Description
This is an easy level CTF for everyone that focuses on encodings, ciphers and hashes. If you are familiar with the terminal, feel free to use it to complete the following tasks. If you're not, don't worry - these websites can prove handy in finding the answers:

CyberChef. Useful when it comes to encodings & ciphers. (If you don't know how to use it, check this room out)

Cryptii. Also pretty great for encodings and encryptions.

Hashes.com. Personally my favourite hash cracking tool.

CrackStation. Another hash cracker you might like.

Why don't you go explore them?

Scenario 
You are a junior cryptanalyst and have recently been hired by TryCrackMe, a secret agency specialising in decoding messages. Before starting your job, TryCrackMe decides to see how skilled you are with a knowledge test split into 3 parts: encodings, ciphers and hashes. Can you complete the challenge and prove your worth?
